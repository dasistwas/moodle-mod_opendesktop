Open Virtual Desktop

Ein Open Virtual Desktop-Handbuch und Anleitungen finden Sie unter OpenDesktop_module documentation

Das Moodle-Modul OpenDesktop ermöglicht eine Einbindung des Ulteo Open Virtual Desktop (OVD) in Moodle.

Wenn Sie eine eigene Open Virtual Desktop - Umgebung verwenden möchten, wird empfohlen den OVD-Application-Server und den OVD-Sessionmanager auf zwei getrennten Servern zu installieren. Sie können versuchen, den Sessionmanager und Moodle (und auch den Application-Server) am selben Rechner laufen zu lassen, beachten Sie aber, dass dies zu Konfigurationsproblemen am Apache-Server führen kann. Wenn Sie dennoch den Sessionmanager und Moodle am selben Server installieren, verwenden Sie als Sessionmanager-URL localhost/sessionmanager.

Alle Infos zur Installation sowie Hilfe bei Fragen finden Sie unter OpenDesktop_module documentation

Die in Moodle verwendeten Rollen gelten auch für das Moodle-Modul OpenDesktop: Teilnehmer/innen und Gäste können bereits gestartete Desktops ihres Kurses ansehen, Lehrer/innen können Desktops starten und an bereits gestarteten Desktops aktiv oder passiv (nur ansehen) teilnehmen.
